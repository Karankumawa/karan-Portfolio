# Karan's Developer Portfolio

A modern, responsive personal portfolio website designed to showcase my web development skills, work experience, and recent projects. Built with clean UI/UX principles in mind.

##  Live Demo
[Click here to view my live portfolio](-------------------)

##  Features
* **Modern & Clean UI:** Designed with a consistent color palette and modern typography (Poppins).
* **Fully Responsive:** Mobile-first approach ensuring the site looks great on phones, tablets, and desktops.
* **Interactive Elements:** Smooth scrolling, hover effects, and a dynamic typing animation in the hero section.
* **Vertical Timeline:** A clean, chronological display of work experience and milestones.
* **Categorized Skills:** A modern "pill" tag layout for technical expertise.

##  Built With
* **HTML5:** Semantic structuring.
* **CSS3:** Custom styling, Flexbox, CSS Grid, and CSS Variables for theming.
* **JavaScript:** Smooth scrolling and DOM manipulation.
* **Typed.js:** Library used for the dynamic typing effect in the hero section.
